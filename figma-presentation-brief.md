# NovaOps Reactor Intelligence Platform — Presentation Brief

## Goal
Create a high-impact presentation that explains the NovaOps Reactor Intelligence Platform, demonstrates real-time monitoring, and positions the product as the premium industrial AI operations system.

## Audience
- Industrial operations leadership
- Process engineers and plant managers
- Digital transformation stakeholders

## Story Arc
1. The problem: legacy monitoring and delayed reaction time
2. The solution: NovaOps real-time digital twin + analytics
3. Proof: KPIs, telemetry, anomaly detection, and safety response
4. Impact: operational efficiency, yield improvement, downtime reduction

## Slide Outline (12–14 Slides)
1. **Title** — NovaOps Reactor Intelligence Platform
   - Background: dark industrial texture, subtle grid, neon accent line
   - Hero visual: reactor silhouette with glow

2. **Problem Statement**
   - 3–4 bullet pain points
   - Visual: degraded telemetry feed, warning overlays

3. **What NovaOps Is**
   - One sentence definition
   - Three pillars: Digital Twin, Live Telemetry, AI Insights

4. **System Architecture**
   - Diagram: Sensors → Edge → MQTT → Analytics → UI

5. **Live Telemetry Dashboard**
   - Full-bleed screenshot mock
   - Callouts for flow, temperature, pressure

6. **Conversion Rate Analytics**
   - Line chart + gauge
   - Status interpretation panel

7. **Equilibrium Drift**
   - Semi-circular gauge + trend chart
   - Advisory copy

8. **Hotspot Formation**
   - Thermal spike chart with red highlights
   - Risk badge state ladder

9. **Digital Twin Response**
   - Reactor core glow overlay
   - Conditional alert animation notes

10. **Anomaly Detection**
   - Isolation Forest KPI grid
   - Timeline chart

11. **Operational Impact**
   - 3 metrics with large typography
   - Example: +4% yield, -18% downtime, +22% safety response speed

12. **Implementation Roadmap**
   - 3-phase timeline: Pilot, Scale, Optimize

13. **Security & Reliability**
   - MQTT over TLS, RBAC, audit logs, data retention

14. **Call to Action**
   - Pilot proposal + contact

## Design System
### Color Palette
- Background: #050505, #0B1220
- Surface: #0F172A with 30–50% opacity
- Primary neon: #00F0FF
- Warning: #FACC15
- Alert: #EF4444
- Success: #4ADE80
- Text: #E2E8F0, #94A3B8

### Typography
- Title: Inter / 52–64, Bold, tracking 2–4%
- Section: Inter / 24–32, SemiBold
- Body: Inter / 14–18, Regular
- Labels: Inter / 10–12, Uppercase, tracking 10–20%
- Numeric: JetBrains Mono / 18–32

### Layout
- Frame size: 1920×1080
- Grid: 12 columns, 80px margins, 24px gutters
- Glassmorphism panels: 12–16px radius, 1px border at 10% white

## Visual Components
- KPI Card: title, value, micro trend
- Chart Panel: title, chart, legend
- Status Badge: Normal / Warning / Critical
- Callout Pills: context labels for charts

## Data Visualization Specs
- Use Recharts-style visuals as reference
- Line thickness: 2px
- Dashed equilibrium: 4/4
- Glow for critical ranges: 10–20px soft blur

## Assets
- Reactor silhouette or clean outline
- Industrial texture overlay (low opacity)
- Icon set: Lucide-style monochrome

## Figma Build Notes
- Create components for KPI cards, chart panels, badges
- Use Auto Layout for responsive alignment
- Keep spacing consistent (8px baseline grid)

## Deliverables
- 1 Figma page “Presentation”
- 12–14 slides fully laid out
- Component library on a separate frame
